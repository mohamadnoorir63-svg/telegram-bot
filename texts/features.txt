<b>🧭 راهنمای مدیریت گروه</b>

✨ <b>🔒 دستورات قفل و باز کردن محتوا</b>
<i>قفل [نوع]</i> → فعال کردن قفل  
<i>باز کردن [نوع]</i> → غیرفعال کردن قفل

📌 <b>انواع قفل‌ها:</b>  
لینک, عکس, ویدیو, فایل, ویس, استیکر, گیف, رسانه, فوروارد, تبلیغ, تبچی, یوزرنیم/منشن, ورود, پیام سرویس, عربی, انگلیسی, متن, آهنگ, ایموجی, کپشن, ریپلای, پیام طولانی, حروف بزرگ, لینک کوتاه, لینک ویدیو, پیام تکراری

💡 <b>مثال‌ها:</b>  
<i>قفل لینک</i> → فعال شدن قفل لینک  
<i>باز کردن عکس</i> → باز کردن قفل عکس

📊 <b>نمایش وضعیت قفل‌ها:</b> <i>/locks</i>

━━━━━━━━━━━━━━━━━━
<b>👑 دستورات VIP</b>
- «تنظیم ویژه» → افزودن کاربر به VIP  
- «حذف ویژه» → حذف کاربر از VIP  
- «لیست ویژه» → نمایش کاربران VIP

━━━━━━━━━━━━━━━━━━
<b>👮 مدیریت کاربران</b>
- ریپلای + بن → مسدودسازی کاربر  
- ریپلای + حذف بن → رفع بن  
- لیست بن → کاربران بن‌شده  
- ریپلای + سکوت → سکوت دائم  
- سکوت [زمان] → سکوت موقت  
- ریپلای + حذف سکوت → رفع سکوت  
- لیست سکوت → کاربران ساکت  
- ریپلای + اخطار → افزودن اخطار  
- ریپلای + حذف اخطار → حذف اخطار  
- لیست اخطار → نمایش همه اخطارها

━━━━━━━━━━━━━━━━━━
<b>👑 مدیریت مدیران</b>
- ریپلای + افزودن مدیر → اضافه کردن مدیر  
- ریپلای + حذف مدیر → حذف مدیر  
- لیست مدیران → نمایش مدیران  
- پاکسازی مدیران → حذف همه مدیران (فقط سودو)

━━━━━━━━━━━━━━━━━━
<b>🧹 پاکسازی پیام‌ها</b>
- پاکسازی → حذف ۱۰۰ پیام اخیر  
- پاکسازی [عدد] → حذف تعداد دلخواه  
- پاکسازی کامل → حذف گسترده  
- ریپلای + پاکسازی → حذف پیام‌های کاربر  
- پاکسازی پیام ربات → حذف پیام‌های ربات‌ها

━━━━━━━━━━━━━━━━━━
<b>📌 پین پیام‌ها</b>
- ریپلای + پین → سنجاق پیام  
- ریپلای + پین [عدد] → پین موقت  
- حذف پین → حذف همه پین‌ها

━━━━━━━━━━━━━━━━━━
<b>👤 لقب و اصل</b>
- ثبت لقب [متن] → تعیین لقب خود  
- ریپلای + ثبت لقب → لقب برای دیگران  
- لقب من / حذف لقب / لیست لقب‌ها  
- ثبت اصل [متن] → تعیین اصل  
- ریپلای + ثبت اصل → اصل برای دیگران  
- اصل من / حذف اصل / لیست اصل‌ها

━━━━━━━━━━━━━━━━━━
<b>🏷 تگ‌ها</b>
- تگ همه → تگ همه اعضا  
- تگ فعال → کاربران فعال  
- تگ غیرفعال → کاربران غیرفعال  
- تگ مدیران → تگ مدیران گروه
<b>🧭 راهنمای کامل قفل‌های گروه</b>

<b>🔒 دستور کلی قفل / باز کردن:</b>  
<i>قفل [نوع]</i> → فعال کردن قفل  
<i>باز کردن [نوع]</i> یا <i>بازکردن [نوع]</i> → غیرفعال کردن قفل

━━━━━━━━━━━━━━━━━━
<b>📌 انواع قفل‌های موجود:</b>

<b>🚫 لینک‌ها:</b>  
- لینک → ارسال هر نوع لینک  
- همه لینک‌ها → همه لینک‌ها (شامل t.me, http, telegram.me)  
- لینک کوتاه → bit.ly, tinyurl, t2m.io  
- لینک ویدیو → youtube, tiktok, youtu.be

<b>📸 رسانه‌ها:</b>  
- عکس → ارسال عکس  
- ویدیو → ارسال ویدیو  
- فایل → ارسال فایل  
- ویس → ارسال ویس  
- استیکر → ارسال استیکر  
- گیف → ارسال گیف  
- رسانه → همه رسانه‌ها  
- فوروارد → فوروارد پیام

<b>🗣 پیام‌ها:</b>  
- متن → پیام متنی  
- کپشن → کپشن‌گذاری ممنوع  
- ریپلای → پاسخ دادن ممنوع  
- پیام طولانی → پیام با بیش از حد مشخص (مثلا 200 کاراکتر)  
- حروف بزرگ → پیام با حروف بزرگ تمام  
- پیام تکراری → ارسال پیام مشابه قبلی

<b>💬 تبلیغات و لینک خارجی:</b>  
- تبلیغ → تبلیغات، joinchat، bot?start، channel  
- تبچی / ربات اینلاین → استفاده از ربات اینلاین ممنوع

<b>🔤 زبان و کاراکتر:</b>  
- عربی → حروف عربی  
- انگلیسی → حروف انگلیسی  
- ایموجی → ارسال فقط ایموجی مجاز نیست

<b>👥 تگ و منشن:</b>  
- تگ → استفاده از @ یا منشن  
- یوزرنیم → استفاده از یوزرنیم

<b>🤖 ورود / ربات‌ها:</b>  
- ورود → ورود کاربران جدید  
- ربات → ارسال پیام توسط ربات

<b>📦 فایل و رسانه پیشرفته:</b>  
- فایل‌های خاص → انواع فایل مشخص  
- رسانه خارجی → رسانه‌ای که از لینک خارجی فرستاده شده  
- فوروارد از ربات → فوروارد پیام از ربات  
- inline bots → ربات‌های اینلاین

━━━━━━━━━━━━━━━━━━
<b>💡 مثال‌های عملی:</b>  
- <i>قفل لینک</i> → فعال شدن قفل لینک  
- <i>قفل عکس</i> → فعال شدن قفل عکس  
- <i>باز کردن ویدیو</i> → غیرفعال شدن قفل ویدیو  
- <i>قفل پیام طولانی</i> → پیام طولانی حذف می‌شود  
- <i>قفل تبلیغ</i> → تبلیغات حذف می‌شود

━━━━━━━━━━━━━━━━━━
