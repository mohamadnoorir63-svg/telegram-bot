📚 <b>راهنمای کامل ربات و مدیریت گروه</b>
────────────────────────

🪪 <b>اصل و لقب</b>
- ثبت اصل: ریپلای روی پیام + <code>ثبت اصل</code>
- ثبت لقب: ریپلای روی پیام + <code>ثبت لقب</code>
- نمایش اصل: ریپلای + <code>اصل</code>
- نمایش لقب: ریپلای + <code>لقب</code>
- اصل خودم: <code>اصل من</code>
- لقب خودم: <code>لقب من</code>
- <code>/rename &lt;nickname&gt;</code> → تغییر لقب خود در گروه
- <code>/shownickname</code> → نمایش لقب فعلی اعضا

────────────────────────

🚫 <b>تنبیه‌ها</b>
- بن: ریپلای یا <code>@id</code> + <code>بن</code>
- <code>/حذف بن</code> → رفع بن
- سکوت: ریپلای یا <code>@id</code> + <code>سکوت [مدت زمان]</code>
- <code>/حذف سکوت</code> → پایان سکوت
- اخطار: ریپلای یا <code>@id</code> + <code>اخطار</code>
- <code>/حذف اخطار</code> → پاک کردن اخطار کاربر

────────────────────────

📌 <b>سنجاق و حذف سنجاق</b>
- پن: ریپلای روی پیام + <code>پن [مدت زمان]</code>
- حذف پن: ریپلای + <code>حذف پن</code> → حذف سنجاق پیام یا همه سنجاق‌ها

────────────────────────

👥 <b>تگ کاربران</b>
- <code>تگ همه</code> → تگ همه اعضا
- <code>تگ مدیران</code> → تگ همه مدیران
- <code>تگ فعال</code> → تگ کاربرانی که ۲۴ ساعت گذشته فعال بودند
- <code>تگ غیرفعال</code> → تگ کاربرانی که ۲۴ ساعت گذشته غیرفعال بودند
- <code>تگ تصادفی [تعداد]</code> → تگ تصادفی اعضای گروه

────────────────────────

🚫 <b>فیلتر کلمات</b>
- <code>فیلتر &lt;کلمه&gt; [مدت زمان]</code> → اضافه کردن کلمه به فیلتر
- <code>حذف فیلتر &lt;کلمه&gt;</code> → حذف کلمه از فیلتر
- <code>لیست فیلتر</code> → نمایش کلمات فیلتر شده
- <code>فیلتر روشن</code> → فعال کردن فیلتر
- <code>فیلتر خاموش</code> → غیرفعال کردن فیلتر

────────────────────────

💾 <b>دستورات شخصی</b>
- <code>/save &lt;name&gt;</code> → ذخیره یک دستور شخصی
- <code>/del &lt;name&gt;</code> → حذف دستور شخصی
- <code>/listcmds</code> → نمایش دستورات ذخیره‌شده

────────────────────────

🎮 <b>پنل و تنظیمات ربات</b>
- <code>پنل</code> → باز کردن منوی اصلی
- <code>لینک</code> یا <code>Link</code> → باز کردن پنل لینک‌ها
- <code>/mode</code> → تغییر حالت ربات
- <code>/welcome</code> → فعال/غیرفعال کردن خوشامدگویی

────────────────────────

🚫 <b>قفل محتوای پیام‌ها</b>
> ⚠️ <b>مثال عملی:</b>
> - <code>قفل لینک</code> → فعال/غیرفعال کردن ارسال لینک
> - <code>باز کردن لینک</code> → باز کردن قفل لینک

────────────────────────

🔒 <b>قفل کل گروه</b>
- <code>قفل گروه</code> → غیر فعال کردن ارسال پیام توسط همه
- <code>باز کردن گروه</code> → باز کردن قفل گروه

────────────────────────

⏰ <b>قفل خودکار گروه</b>
- <code>تنظیم قفل خودکار &lt;ساعت شروع&gt; &lt;ساعت پایان&gt;</code> → تنظیم بازه زمانی قفل خودکار
- <code>قفل خودکار روشن</code> → فعال کردن قفل خودکار گروه
- <code>قفل خودکار خاموش</code> → خاموش کردن قفل خودکار گروه

────────────────────────

🧹 <b>دستورات پاکسازی پیام‌ها</b>
- <code>پاکسازی</code> / <code>clean</code> → حذف همه پیام‌ها
- <code>حذف &lt;تعداد&gt;</code> / <code>پاک &lt;تعداد&gt;</code> → حذف n پیام آخر
- ریپلای روی پیام کاربر + <code>پاک</code> / <code>حذف</code> → حذف پیام‌های آن کاربر

────────────────────────

🔁 <code>/reply</code> → فعال/غیرفعال کردن پاسخ خودکار

────────────────────────

🛠 <b>توضیحات</b>
- مدیران و سودوها از همه قفل‌ها معاف هستند  
- قفل خودکار توسط <code>JobQueue</code> هر دقیقه بررسی می‌شود  
- دستورات قفل محتوایی فقط برای مدیران و سودوها قابل تغییر است  
- لقب‌ها و تغییر اسم‌ها تنها برای اعضای عادی قابل استفاده است  

────────────────────────
