📚 راهنمای کامل ربات  و مدیریت گروه
────────────────────────

🪪 اصل و لقب  
- ثبت اصل: ریپلای روی پیام + "ثبت اصل"  
- ثبت لقب: ریپلای روی پیام + "ثبت لقب"  
- نمایش اصل: ریپلای + "اصل"  
- نمایش لقب: ریپلای + "لقب"  
- اصل خودم: "اصل من"  
- لقب خودم: "لقب من"  
- /rename <nickname> → تغییر لقب خود در گروه  
- /shownickname → نمایش لقب فعلی اعضا  

────────────────────────

🚫 تنبیه‌ها  
- بن: ریپلای یا @id + بن  
- /حذف بن → رفع بن  
- سکوت: ریپلای یا @id + سکوت [مدت زمان]  
- /حذف سکوت → پایان سکوت  
- اخطار: ریپلای یا @id + اخطار  
- /حذف اخطار → پاک کردن اخطار کاربر  

────────────────────────

📌 سنجاق و حذف سنجاق  
- پن: ریپلای روی پیام + "پن [مدت زمان]"  
- حذف پن: ریپلای + "حذف پن" → حذف سنجاق پیام یا همه سنجاق‌ها  

────────────────────────

👥 تگ کاربران  
- تگ همه → تگ همه اعضا  
- تگ مدیران → تگ همه مدیران  
- تگ فعال → تگ کاربرانی که ۲۴ ساعت گذشته فعال بودند  
- تگ غیرفعال → تگ کاربرانی که ۲۴ ساعت گذشته غیرفعال بودند  
- تگ تصادفی [تعداد] → تگ تصادفی اعضای گروه  

────────────────────────

🚫 فیلتر کلمات  
- فیلتر <کلمه> [مدت زمان] → اضافه کردن کلمه به فیلتر  
- حذف فیلتر <کلمه> → حذف کلمه از فیلتر  
- لیست فیلتر → نمایش کلمات فیلتر شده  
- فیلتر روشن → فعال کردن فیلتر  
- فیلتر خاموش → غیرفعال کردن فیلتر  

────────────────────────

💾 دستورات شخصی  
- /save <name> → ذخیره یک دستور شخصی  
- /del <name> → حذف دستور شخصی  
- /listcmds → نمایش دستورات ذخیره‌شده  

────────────────────────

🎮 پنل و تنظیمات ربات  
- پنل → باز کردن منوی اصلی  
- لینک یا Link → باز کردن پنل لینک‌ها  
- /mode → تغییر حالت ربات  
- /welcome → فعال/غیرفعال کردن خوشامدگویی  

────────────────────────

🚫 قفل محتوای پیام‌ها (نمونه)  
> ⚠️ مثال عملی:  
> - قفل لینک → فعال/غیرفعال کردن ارسال لینک  
> - باز کردن لینک → باز کردن قفل لینک  

لیست کامل انواع قفل‌ها:  
- قفل عکس  
- قفل ویدیو  
- قفل فایل  
- قفل ویس  
- قفل استیکر  
- قفل گیف  
- قفل فوروارد  
- قفل لینک  
- قفل تبلیغ  
- قفل یوزرنیم  
- قفل منشن  
- قفل متن  
- قفل ایموجی  
- قفل کپشن  
- قفل ریپلای  
- قفل عربی  
- قفل انگلیسی  
- قفل موزیک  
- قفل ویس چت  
- قفل مکان  
- قفل مخاطب  
- قفل نظرسنجی  
- قفل ربات  
- قفل ورود  

────────────────────────

🔒 قفل کل گروه  
- قفل گروه → غیر فعال کردن ارسال پیام توسط همه  
- باز کردن گروه → باز کردن قفل گروه  

────────────────────────

⏰ قفل خودکار گروه  
- تنظیم قفل خودکار <ساعت شروع> <ساعت پایان> → تنظیم بازه زمانی قفل خودکار  
- قفل خودکار روشن → فعال کردن قفل خودکار گروه  
- قفل خودکار خاموش → خاموش کردن قفل خودکار گروه  

────────────────────────

🧹 دستورات پاکسازی پیام‌ها  
- پاکسازی / clean → حذف همه پیام‌ها  
- حذف <تعداد> / پاک <تعداد> → حذف n پیام آخر  
- ریپلای روی پیام کاربر + پاک / حذف → حذف پیام‌های آن کاربر  

────────────────────────

🔁 /reply → فعال/غیرفعال کردن پاسخ خودکار  

────────────────────────

🛠 توضیحات  
- مدیران و سودوها از همه قفل‌ها معاف هستند  
- قفل خودکار توسط JobQueue هر دقیقه بررسی می‌شود  
- دستورات قفل محتوایی فقط برای مدیران و سودوها قابل تغییر است  
- لقب‌ها و تغییر اسم‌ها تنها برای اعضای عادی قابل استفاده است
